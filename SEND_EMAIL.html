SEND_EMAIL

Le SEND_EMAIL est un logiciel qui vous permet

d'envoyer un email ( avec une option de spam)

anonymement ou normalement






Envoyer un e-mail anonymement :


[+] étape 1 :

Inserer l'e-mail de la victime.



[+] étape 2 :

Entrer le nombre de fois que

vous voulez que le message s'envoie



[+] étape 3 :

Entrer l'objet que vous voulez mettre



[+] étape 4 :

Entrer le message que vous voulez envoyer



[+] étape 5 :

Si le message a correctement était envoyer

il y aura le même message

que dans le screen si dessous





Envoyer un emails normale :


[+] étape 1 :

Insérer votre e-mail



[+] étape 2 :

Entrer l'app password de votre e-mail



[+] étape 3 :

Entrer l'e-mail de la personne



[+] étape 4 :

Entrer le nombre de fois que le message s'envoie



[+] étape 5 :

Entrer l'objet que vous souhaitez mettre



[+] étape 6 :

Si le message a correctement était envoyer

il y aura le même message

que dans le screen si dessous
